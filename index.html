Type Provider library for 101explorer

This library provides types that can be used to access 101explorer. E.g. for accessing a specific file:

let url = "http://101companies.org/resources/contributions/javaStatic/src/test/java/org/softlang/company/tests/CutTest.java"
let target = OneOhOne.File(url)

printfn "%s" target.classifier
printfn "%s" target.language
target.derived |> Seq.iter( fun derived ->
    printfn "%s" derived.name
)

How does it work

The types are dynamically generated based on the JSON schemas that can also be found on the 101worker repository. The actual data is delivered by 101explorer.

So, for example the file schema:

        "classifier": {
            "type": "string",
            "description": "classifier of this entity"
        },
        "geshi": {
            "type": "string",
            "description": "geshi code that can be used for highlighting"
        },
        "derived": {
            "type": "array",
            "description": "array of all derived files",
            "items": {
